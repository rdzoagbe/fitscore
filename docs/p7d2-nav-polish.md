# P7D.2 Navigation Polish

This patch adds a dedicated navigation styling layer. It does not change routes, API logic, Supabase, Stripe, or authentication.

## What to check

- Top nav is less cramped on desktop.
- Active tab is clearly visible.
- Buttons have consistent height and radius.
- Mobile/tablet nav scrolls horizontally rather than breaking layout.
- Brand text does not dominate small screens.
- No horizontal overflow outside the nav.
