# P6C Visual Copy System — Joblytics AI

## Product promise

**Know what to improve before you apply.**

Joblytics should not sound like a generic AI text box. It should sound like a practical job-search workspace that turns a CV and a job description into an application plan.

## Preferred language

Use phrases such as:

- Application readiness
- Role-specific application plan
- Proof gaps recruiters will test
- Keywords to surface with evidence
- Next best action before applying
- From CV check to interview prep

## Avoid

Avoid phrases such as:

- AI-powered magic
- Guaranteed interviews
- Beat the ATS
- Perfect CV in seconds
- Automatic LinkedIn scraping
- Scrape LinkedIn profile URL

## CTA hierarchy

Primary CTA:

- Start a free ATS check

Secondary CTAs:

- Check my CV against a job
- Optimize my LinkedIn profile
- Organize my applications
- Compare plans

## Trust language

Use this wording when a page mentions LinkedIn, CV uploads, or saved history:

> No LinkedIn password. No private page scraping. Keep your career material inside your workspace and decide what to save.

## Visual copy rules

1. Put the user's decision first: apply, improve first, or skip.
2. Use concrete nouns: CV, job description, proof gaps, keywords, cover letter, interview kit.
3. Avoid vague claims like “better results.” Say what improves and where.
4. Keep public SEO pages useful even before login.
5. Make CTAs action-based, not hype-based.
