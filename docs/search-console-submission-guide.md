# Search Console Submission Guide

## Add property

1. Open Google Search Console.
2. Add property: `https://joblytics-ai.com/`
3. Verify domain ownership using the method Google provides.

## Submit sitemap

Submit:

```text
https://joblytics-ai.com/sitemap.xml
```

## Check these sections

- Security issues
- Manual actions
- Pages / indexing
- Sitemaps
- URL inspection

## URL inspection list

Inspect these first:

- https://joblytics-ai.com/
- https://joblytics-ai.com/pricing
- https://joblytics-ai.com/resources
- https://joblytics-ai.com/roles
- https://joblytics-ai.com/sample-reports
- https://joblytics-ai.com/early-access
- https://joblytics-ai.com/privacy

## Request indexing

For priority public pages, use URL Inspection → Request indexing.

## Do not request indexing for

- /admin
- /admin/reliability
- /api/*
- private authenticated app states
