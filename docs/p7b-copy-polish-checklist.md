# P7B Copy Polish Checklist

Use this checklist to remove generic AI copy and make Joblytics sound like a serious career product.

## Preferred product language

Use phrases like:

- Application readiness
- Role-specific application plan
- Proof points recruiters will look for
- Missing keywords and missing evidence
- Before you apply
- Improve first / Apply / Skip
- Private CV and profile analysis
- Save your application history

Avoid phrases like:

- Supercharge your career
- AI-powered magic
- Instantly land your dream job
- Guaranteed interviews
- Scrape LinkedIn
- Connect LinkedIn
- LinkedIn login

## Page-by-page copy standards

### Homepage

- [ ] Hero explains the outcome in one sentence.
- [ ] CTA is action-oriented: “Run a free ATS check”.
- [ ] Product preview uses concrete results: score, missing keywords, next action.
- [ ] Trust block mentions private data and paste-only LinkedIn mode.

### Pricing

- [ ] Payment buttons do not look broken while Stripe is not live.
- [ ] Copy says checkout is being prepared.
- [ ] Early-access/contact CTA is clear.

### LinkedIn Optimizer

- [ ] Paste-only mode is explicit.
- [ ] No LinkedIn credential, login, import, scraping, or profile URL claims.
- [ ] The page says users choose what they paste.

### Dashboard

- [ ] Workflow is written like a career plan, not a tool list.
- [ ] Next best action tells the user what to do next and why.

### SEO pages

- [ ] Avoid duplicated paragraphs across pages.
- [ ] Each role page speaks to the role-specific pain point.
- [ ] Every public page has one clear CTA.
